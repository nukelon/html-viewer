# html-viewer

可直接部署到 GitHub Pages 的在线网页预览器。

## 功能

- 上传任意文件，支持文件夹上传；若当前为空，首次上传文件夹会自动把文件夹内内容平铺到根目录。
- 若当前文件管理器为空且只上传一个 ZIP，会自动解压到根目录并保留内部结构。
- 输入 HTML 文件名（需带扩展名）或 URL 后点击“运行”；留空默认运行 `index.html`。
- 预览成功后，文件管理器变为左侧边栏，右侧显示网页内容，并展示 Safari 风格顶部蓝色加载进度条。
- 可拖拽中间分隔条调整比例，范围为 `1/8` 到 `3/2`（文件管理器 : 预览区）。

## 本地运行

直接使用静态服务器即可，例如：

```bash
python3 -m http.server 8000
```
